# 《天下要闻｜A股映射日报》数据增强评分模型

更新时间：2026-03-28

## 一、目标
在新闻映射基础上，为个股 / 期货 / 商品增加一个更可复用的判断框架，尽量输出“更偏高胜率”的方向，而不是只输出“高相关”。

---

## 二、评分维度（总分 100）

### 1. 新闻逻辑强度（30分）
- 30：全球主线 / 一级风险事件 / 明确政策催化
- 20：重要分支主线
- 10：弱题材联想
- 0：基本无映射基础

### 2. 资产走势共振（25分）
- 25：价格与新闻方向高度共振
- 15：价格有一定共振，但节奏不稳
- 5：逻辑与价格背离
- 0：完全无共振

### 3. A股板块承接（20分）
- 20：板块强势且有资金承接
- 10：板块有反应但不稳定
- 0：板块无明显承接

### 4. 标的辨识度（15分）
- 15：龙头 / 高辨识度 / 逻辑直接
- 8：次核心标的
- 3：弱联想标的
- 0：映射牵强

### 5. 风险扣分（-10 至 0）
扣分项包括：
- 题材过热
- 连续大涨后性价比差
- 数据与逻辑明显背离
- 非交易时段信息不完整

---

## 三、评分结果解释
- **80分以上**：更偏高胜率方向
- **60-79分**：高相关但更看节奏
- **40-59分**：观察方向
- **40分以下**：不建议作为重点映射输出

---

## 四、默认输出形式
每次若启用数据增强，可在标的后附：
- 评分：85 / 100
- 结论：更偏高胜率方向

例如：
- 中国海油：85 / 100（更偏高胜率）
- 中远海能：68 / 100（高相关但更看节奏）
- 中航沈飞：62 / 100（高相关但更依赖情绪强化）

---

## 五、执行原则
- 分数只是辅助，不替代判断
- 必须和文字解释同时出现
- 禁止只报分数、不解释依据
